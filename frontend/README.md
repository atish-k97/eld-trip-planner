# ELD Trip Planner — Frontend

React + Vite frontend for the ELD Trip Planner app.

**Live Demo:** https://eld-trip-planner-azure.vercel.app

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local`:

```
VITE_API_URL=https://eld-trip-planner-8ye2.onrender.com
```
